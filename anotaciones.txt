Formas de aplicar CSS
1. En linea: Se aplica directamente en la etiqueta utilizando el atributo style. En el valor del atributo style se escribe sintaxis CSS como la mostrada a continuacion:

    nombre-estilo1:valor-estilo1;
    nombre-estilo2:valor-estilo2;
    ...
    nombre-estiloN:valor-estiloN;

2. En el encabezado del html: Se incluye estilos utilizando la etiqueta style dentro de la etiqueta head.
    Para aplicar este tipo de estilos a una etiqueta en particular es necesario utilizar selectores.
    Tipos de selectores:
    *Selector por nombre de etiqueta:
        nombre-etiqueta{
            nombre-estilo1:valor-estilo1;
            nombre-estilo2:valor-estilo2;
            ...
            nombre-estiloN:valor-estiloN;
        }
    *Selector por el ID de la etiqueta:
        #id-etiqueta{
            nombre-estilo1:valor-estilo1;
            nombre-estilo2:valor-estilo2;
            ...
            nombre-estiloN:valor-estiloN;
        }
    *Selector por clase CSS
        .nombre-clase{
            nombre-estilo1:valor-estilo1;
            nombre-estilo2:valor-estilo2;
            ...
            nombre-estiloN:valor-estiloN;
        }
